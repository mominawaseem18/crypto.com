# 🚀 NEFA Crypto Landing Page

A modern and responsive cryptocurrency landing page built using HTML5 and CSS3. This project recreates the NEFA crypto website hero section with a clean navigation bar, call-to-action buttons, and an attractive user interface.

## 📸 Features

* Responsive Navigation Bar
* Modern Hero Section
* Custom Logo Design
* Call-to-Action Buttons
* Clean and Minimal UI
* HTML5 Semantic Structure
* CSS3 Flexbox Layout

## 🛠️ Technologies Used

* HTML5
* CSS3

## 📂 Project Structure

```text
├── crypto.html
├── crypto.css
└── assets/
    └── crypto-image.png
```

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/your-username/nefa-crypto-landing-page.git
```

2. Open the project folder

3. Run `crypto.html` in your browser
